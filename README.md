# Graph Algorithms API

A Java graph library implementing core graph data structures and algorithms for both directed and undirected graphs.

The project demonstrates:
- generic type-safe abstractions for vertices, edges, and graph payloads
- custom data structures, including a self-implemented priority queue
- graph traversal algorithms: BFS and DFS
- shortest-path search with Dijkstra’s algorithm
- topological sorting with Kahn’s algorithm
- minimum spanning tree construction with Prim’s algorithm
- robust handling of edge cases such as missing nodes, cycles, and disconnected graphs

Built with a clean package structure and designed for extensibility and reuse in bigger projects.
